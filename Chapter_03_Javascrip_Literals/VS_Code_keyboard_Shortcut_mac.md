# VS Code Keyboard Shortcuts - macOS

## General

| Shortcut | Action |
|----------|--------|
| `Cmd + Shift + P` | Show Command Palette |
| `Cmd + P` | Quick Open (Go to File) |
| `Cmd + Shift + N` | New Window |
| `Cmd + W` | Close Window |
| `Cmd + ,` | Open User Settings |
| `Cmd + K Cmd + S` | Open Keyboard Shortcuts |

## Basic Editing

| Shortcut | Action |
|----------|--------|
| `Cmd + X` | Cut line (empty selection) |
| `Cmd + C` | Copy line (empty selection) |
| `Cmd + V` | Paste |
| `Cmd + Shift + K` | Delete Line |
| `Cmd + Enter` | Insert Line Below |
| `Cmd + Shift + Enter` | Insert Line Above |
| `Cmd + Shift + \` | Jump to Matching Bracket |
| `Cmd + ]` | Indent Line |
| `Cmd + [` | Outdent Line |
| `Home` | Go to Beginning of Line |
| `End` | Go to End of Line |
| `Cmd + Home` | Go to Beginning of File |
| `Cmd + End` | Go to End of File |
| `Option + Up` | Move Line Up |
| `Option + Down` | Move Line Down |
| `Option + Shift + Up` | Copy Line Up |
| `Option + Shift + Down` | Copy Line Down |
| `Cmd + /` | Toggle Line Comment |
| `Option + Shift + A` | Toggle Block Comment |
| `Cmd + K Cmd + C` | Add Line Comment |
| `Cmd + K Cmd + U` | Remove Line Comment |
