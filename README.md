## ЁЯЪА Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) installed on your machine
- [VS Code](https://code.visualstudio.com/) (recommended editor)
- Basic understanding of how to open a terminal/command prompt

### Running the Files

1. **Clone the repository**
   ```bash
   git clone https://github.com/NiskAutomation/LearnPlaywrightBatch2X.git
   cd LearnPlaywrightBatch2X
   ```

2. **Run any JavaScript file using Node.js**
   ```bash
   node Chapter_01_Basics/01_Basics.js
   ```

## ЁЯЫая╕П Naming Conventions Covered

| Case Type | Example | Usage |
|-----------|---------|-------|
| **camelCase** | `userName`, `totalPrice` | Variables, functions |
| **PascalCase** | `UserProfile`, `ShoppingCart` | Classes, constructors |
| **snake_case** | `user_name`, `total_price` | Variables, functions (Python style) |
| **SCREAMING_SNAKE_CASE** | `MAX_SIZE`, `API_KEY` | Constants |
| **camelCase with Acronym** | `htmlParser`, `jsonData` | Variables with acronyms |
| **Hungarian Notation** | `strName`, `bActive`, `nCount` | Type-prefixed (legacy style) |
| **kebab-case** | `my-script-file.js` | File names, strings, CSS classes |
| **dot.case** | `app.name`, `app.version` | Config keys, namespaces |
